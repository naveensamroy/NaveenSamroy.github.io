2. manifest.json Create this new file and place it in the same folder as your index.html. It handles the full-screen display and app icon. { "name": "Samroy Cricket Counter", "short_name": "Cricket", "start_url": "./", "display": "standalone", "background_color": "#000000", "theme_color": "#f1c40f", "icons": [ { "src": "icon-192.png", "sizes": "192x192", "type": "image/png" }, { "src": "icon-512.png", "sizes": "512x512", "type": "image/png" } ] } 3. Icon Files You'll need two icon files. You can create them yourself or use simple, placeholder images. * icon-192.png: A 192x192 pixel image. * icon-512.png: A 512x512 pixel image. Place both of these in the same folder as the other files. Once you upload all three files to your GitHub repository and activate GitHub Pages, your app will be ready to be added to a mobile home screen as a standalone application.
